#AngularWebapp
This is a sample to demo the angular folder structure.  
##Install
`git clone git@github.com:kaiqigong/AngularWebapp.git`  
`cd AngularWebapp`  
`npm install`  
`bower install`  
##Run Preview
`cd app`  
`node ../scripts/web-server.js`  

open `localhost:8000/index.html` in browser.
